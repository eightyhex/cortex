---
description: Find stale notes that need review, archival, or categorization
---

Use the `mcp_staleness_review` tool to detect stale notes in the user's Cortex vault.

$ARGUMENTS

Present results sorted by staleness score. For each note, show the title, type, score, reasons, and suggested action (archive, categorize, or review). Help the user act on each suggestion using `archive_note`, `edit_note`, or other tools as appropriate.
